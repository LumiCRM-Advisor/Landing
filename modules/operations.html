Операционка

Табель учёта рабочего времени, реестр зарплат и управление сотрудниками на объектах — всё в одном окне. Excel-импорт за секунду, расчёт нормочасов и переработок автоматически.

Подключить табель ← Назад к модулям
Возможности
Учёт времени и зарплат без Excel-хаоса
Для аутстаффинговых компаний и вахтовых объектов с регулярными расчётами зарплат. Табель, нормочасы, маржа — всё считается само.
01

Табель с нормочасами

Ежедневный учёт рабочих часов по каждому сотруднику. Автоматический расчёт переработки и недоработки относительно нормы по производственному календарю.

02

Реестр зарплат по объектам

Сводный реестр всех объектов за период с итогами: начислено, к выплате, аванс, маржа клиента. Экспорт в Excel или PDF одним кликом.

03

Excel-импорт и дедупликация

Загрузите список сотрудников из Excel — система проверит дубли и автоматически заполнит карточки. Формат обычный: ФИО, табельный номер, должность.

04

Группировка по объектам

Реестр разбит по объектам и бригадам. Переключение между объектами мгновенное. Итоговая строка показывает данные по всем объектам за период.

05

Типы отсутствия

При нулевых часах автоматически указывается причина: больничный, отпуск, прогул. Влияет на расчёт зарплаты — система учитывает тип отсутствия.

06

Аванс и итоговые выплаты

Учёт авансовых выплат с янтарным маркером в реестре. Итоговая строка показывает сумму к выплате с учётом аванса — без ошибок двойного расчёта.

07

Маржа и рентабельность

Ставка клиента и маржа отображаются в реестре рядом с выплатой сотрудника. Видите прибыльность каждого объекта без отдельных расчётов.

08

История движения кадров

Все изменения по сотруднику фиксируются: переводы между объектами, изменения ставки, найм и увольнение. Полный audit log с датами и исполнителем.

🏗️

Строительные подрядчики

Переменный состав бригад, несколько объектов параллельно. Табель закрывается по факту выполнения этапа.

⛽

Нефтяные и газовые вахты

Строгий учёт часов на удалённых объектах, ротация через 15/30 дней. Стыковка с перевахтовкой критична.

⛏️

Горнодобывающие предприятия

Сменная работа 24/7, бригадный подряд, особые условия переработки. Нужна история по каждому смен-работнику.

🔧

ЖКХ и эксплуатация объектов

Постоянный персонал, разбросанный по объектам города. Учёт выходных и праздников по производственному календарю.

🛡️

Охранные предприятия

Сменный график 12/24, подменные сотрудники, суточные дежурства. Каждый объект — своя документация.

🧹

Клининг и технический сервис

Мобильные бригады, почасовая оплата, объекты меняются. Быстрый импорт из Excel — главное требование.

Как это работает
Три шага до закрытия расчётного периода
01
📥

Загрузить сотрудников

Импортируйте список из Excel или добавьте вручную. Укажите объект, должность, ставку. Система проверит дубли и сформирует табель на текущий месяц.

→
02
⏱️

Заполнить табель

Введите рабочие часы по каждому дню для каждого сотрудника. Можно копировать предыдущий период. Нормочасы и переработки считаются автоматически.

→
03
📊

Закрыть реестр

Система рассчитает итоговые суммы с учётом ставок, аванса и нормочасов. Экспортируйте реестр в Excel или PDF — готово к передаче в бухгалтерию.
